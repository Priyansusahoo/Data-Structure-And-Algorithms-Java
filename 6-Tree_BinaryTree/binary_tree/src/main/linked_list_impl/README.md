#### Binary Tree - Traversal
    - Depth First Search
        - Preorder Traversal // Root Node -> Left SubTree -> Right SubTree
        - Inorder Traversal // left node -> root node -> right node
        - Post order Traversal
    
    - Breadth First Search
        - Level order Traversal
